College Event Management System
📖 Project Description

The College Event Management System is a responsive web-based project developed to manage and display college events effectively.

This system provides a simple and user-friendly interface where students can view event details and organizers can showcase upcoming events.

💻 Technologies Used

HTML5

CSS3 (Both Internal and External CSS)

Bootstrap 5

⚙️ Styling Structure Explanation

In this project:

HTML is used for the overall structure of the website.

Around 25% of the CSS is written in a separate external CSS file (style.css).

The remaining CSS styling is written inside the HTML file using the <style> tag (internal CSS).

Bootstrap 5 is integrated using CDN inside the HTML file for responsive design.

Because a significant portion of styling is written inside the HTML file, GitHub shows a higher percentage of HTML in the language statistics.

🚀 How to Run the Project

Download or clone the repository.

Keep index.html and style.css in the same folder.

Open index.html in any web browser.

The project will run successfully.
